# Slash-Quest
A simple 2d platformer created using unity
# Controls
- <kbd>↑</kbd> <kbd>↓</kbd> <kbd>←</kbd> <kbd>→</kbd> Move
- <kbd>Z</kbd> Melee attack
- <kbd>Crtl</kbd> Ranged attack
- <kbd>Space</kbd> Jump
- <kbd>Q</kbd> Toggle weapon
- <kbd>Shift</kbd> Run
